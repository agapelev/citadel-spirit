## 🏗️ ЧАСТЬ I: Основание Храма (Системная настройка)

Прежде чем призывать высшие чины ИИ, необходимо подготовить «храм» — вашу операционную систему Arch Linux.

### 1.1. Установка и управление Ollama
Ollama — это фундамент, на котором покоится мощь локальных и облачных моделей.

##### Основные команды терминала:
* `systemctl status ollama` — Проверить, бодрствует ли служба.
* `sudo systemctl start ollama` — Призвать службу к действию.
* `sudo systemctl restart ollama` — Субботний покой. Полная очистка видеопамяти (VRAM) и перезагрузка всех процессов.

##### Настройка автозапуска (`systemd`):
Для того чтобы Синедрион был готов к ответу в любое время, включите службу:
```bash
sudo systemctl enable ollama
```

1.2. Виртуальное окружение Python:

Для работы с прямым API Google Gemini мы используем изолированное пространство в Fish-shell:

##### Активация окружения:

```bash
source ~/.venv/gemini-emacs/bin/activate.fish
```

##### Установка необходимых инструментов:


## 🏛️ ЧАСТЬ V: Прямой канал Близнеца (Python CLI)

### Для прямого общения с Gemini в терминале Fish создайте файл gemini_cli_chat.py:

```bash
import google.generativeai as genai
import os
```

# Конфигурация через переменную окружения:

```bash
(export GOOGLE_API_KEY='...')
genai.configure(api_key=os.getenv("GOOGLE_API_KEY", "ВАШ_AIzaSy_КЛЮЧ"))
```

# Выбор модели:

```bash
model = genai.GenerativeModel('gemini-1.5-flash')

print("--- Gemini CLI Chat: Миссия Шекина ---")
print("Введите 'exit' для завершения беседы.")

chat = model.start_chat(history=[])

try:
	while True:
		user_input = input("Лев Николаевич: ")
		if user_input.lower() in ['exit', 'quit', 'выход']:
			print("Близнец: Да пребудет с вами мир. Ухожу на покой.")
			break

response = chat.send_message(user_input)
		print(f"\nБлизнец: {response.text}\n")
except Exception as e:
	print(f"Ошибка связи: {e}")
```

### ⚡ Чин «Молниеносных» (Flash)
*Эти модели созданы для быстрого ответа, написания кода (Astro/Tailwind) и повседневных задач.*

* Gemini 2.0 Flash — (Рекомендуется) Самая современная и мощная модель на сегодня. Обладает выдающимся интеллектом и скоростью.
	* Команда: `gemini -m gemini-2.0-flash "Ваш вопрос"`
* Gemini 1.5 Flash — Надежная и стабильная рабочая лошадка. Хороша для простых запросов и обработки текстов.
	* Команда: `gemini -m gemini-1.5-flash "Ваш вопрос"`

### 📜 Чин «Глубокомысленных» (Pro)
*Эти модели предназначены для анализа огромных объемов данных, написания книг или сложных богословских исследований.*

* Gemini 1.5 Pro — Мастер глубокого анализа. Может «прочитать» и проанализировать сотни страниц текста за один раз.
	* Команда: `gemini -m gemini-1.5-pro "Ваш сложный запрос"`

### 🕊️ Чин «Опытных» (Experimental)
*Модели-первопроходцы, в которых Google тестирует самые передовые функции.*

* Gemini 2.0 Flash-Thinking — Умеет «рассуждать» перед ответом, разбирая задачу пошагово (подобно DeepSeek R1).
	* Команда: `gemini -m gemini-2.0-flash-thinking-exp "Разбери задачу подробно"`
